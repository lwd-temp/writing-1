金鱼缸，Fish Tank。

从泰国金鱼缸讲起吧。

外面，是灯红酒绿，嫖客喝茶聊天。里面，是圆形监狱。

当谈起嫖娼，嫖客们各有说辞。

有推诿的：老婆满足不了我。

有立牌坊的：我只是图个爽快，老婆是生活伴侣，我们同甘共苦。

说者理直气壮，听者频频点头。假如说话者转换性别，社会想必不会允许女性给自己找这样的理由。偏偏，“理”这个东西，多是由男性定义的。比如，骂人多用“婊子”，绝少用“嫖客”。

嫖客中不乏外国面孔，多是游客。有讨价还价的，也有仗着一口带不利索的英文，端起老爷架子的。这种夸赞别人漂亮，并要对方笑的路数，竟变成了各国男性通用的语言。

本片是全球化三部曲纪录片中的最后一部。是的，嫖娼是个全球化的“生意”。中国也不例外，关于泰国“浴室”，网上攻略遍布，笔法淫秽下作。

纪录片拍摄于十年前。早在那时，金鱼缸的管理就已经标准化，流程化，工业化了。点名，在泰王照片前打卡，记班，拿工牌。好像是工厂，或是企业。

《用后即弃的人》里提到，泰国地多丰饶，唯有北方山区贫瘠，泰国城市里的性工作者便多来自此。山区古称兰纳，于五百年前亡国，几经转手，现分属于中缅老泰。在中国的部分，就是西双版纳，在泰国的部分，就是泰国西北省。

泰国妓院采用的是层层剥削的方式，投资人包给承包商，承包商包给妓院老板，妓院老板再包给皮条客。很多时候，投资人甚至不用踏进妓院一步，有大量华裔，华人，参与其中。

金鱼缸的运营，是专业化，流程化的。皮条客介绍性工作者，如同在推销商品。买单付账，可以刷卡。当时 Windows 7 才刚刚发售，前台用的还 Windows XP 系统。甚至有一个金身佛像，叫人一瞬间，误以为这边是餐馆。

有一个颇有意思的事情，女性性工作者挣钱之后，有时会去男性性工作者。这点我不太知道该如何评论，要留待读者点评了。

夜色已深。鱼缸馆外，狗影重重。外边，有人通宵达旦，有人一夜无眠。

孟加拉

街上没有年轻女人。在男人主导的街道，小贩可以公开大方的说，我在妓院有三个女朋友。

孟加拉信奉伊斯兰教。但说到对宗教的虔诚，嫖客有时还不如性工作者，可能是苦难的生活总是要有些寄托。另外，这个门口跨火的仪式我不太懂，似乎并不是伊斯兰教，大概是地方信仰。

地板破裂之后，泥水从墙根上渗出来。墙上的发霉的霉斑，有时候用纸糊上，有时候来不及糊。性工作者的工作不但是超强度的，还必须小心翼翼，以免在客户中落下恶名。

老鸨榨取着手下一大部分收入，买车买房。孩子和母亲住在一起，有客人的时候，小孩只能睡在外边。

暮色降临，棚户屋里，异色灯光亮起。细看，熙熙攘攘，细听，她在说：为什么受苦的总是女人？为什么我们只有这条路可走？

墨西哥

墨西哥的街道，破败中带着西班牙风情。

白天，红灯区里，来了传教士，围了一圈人作法。晚上，性工作者们虔诚供奉着墨西哥的死亡圣神。

我们或许有太多对于性工作者的好奇了，却甚少将目光聚焦到嫖客身上。这是一群怎么样的人呢？他们会公开的说，对未成年人有兴趣。他们会开车在红灯区咒骂一通，指指点点，然后离开。

他们会鄙夷的说，快看，那些妓女。

殊不知，没有嫖客，哪里来的妓女。
